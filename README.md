# 🎯 AI Object Detection and Tracking

A real-time object detection and tracking application built using YOLOv8, OpenCV, and Streamlit.

## 🚀 Features

- Real-time object detection using YOLOv8
- Object tracking
- Webcam detection
- Video file detection
- Confidence threshold control

## 🛠️ Technologies Used

- Python
- Streamlit
- OpenCV
- YOLOv8 (Ultralytics)
